Supported passports & destinations

VisaStay supports 32+ passport nationalities. Every passport gets the same full coverage — all 157 destination countries, with the exact entry type and maximum stay for that specific passport/destination pair.

How passport coverage works

Set your passport once during onboarding (or add a second trip profile for a different passport). For every destination you search, VisaStay shows the exact entry type — visa exempt, visa on arrival, ETA, eVisa, or visa required — and the maximum stay allowed for your specific passport, not a generic average. See the FAQ for what each entry type means.
